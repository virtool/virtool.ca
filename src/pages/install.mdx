# Install Virtool

## Docker Compose

<Note color="blue">

This replaces the original installation method based on downloading and unpacking
`virtool.tar.gz`.

</Note>

1. [Install Docker](https://docs.docker.com/engine/install/ubuntu/#install-using-the-repository) and
   [the compose plugin](https://docs.docker.com/compose/install/linux/#install-using-the-repository) on your Linux host.
2. Write a `docker-compose.yml` similar to the following.

   This will run Virtool and MongoDB in containers and persist their data in volumes.

   ```yaml
   version: "3.1"
   services:
     mongo:
       image: mongo:4.0
       volumes:
         - mongo:/data/db
     virtool:
       image: ghcr.io/virtool/virtool:4.3.2
       environment:
         VT_DATA_PATH: "/data"
         VT_HOST: "0.0.0.0"
       ports:
         - "9950:9950"
       volumes:
         - virtool:/data
   volumes:
     mongo:
     virtool:
   ```

3. Start the services with:

   ```shell
   docker compose -p virtool up -d
   ```

   Virtool should be available on your host machine at [http://localhost:9950](http://localhost:9950).
